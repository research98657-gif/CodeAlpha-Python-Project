# CodeAlpha-Python-Project
## Python Internship Project 

This repository contains Python projects completed as part of the CodeAlpha Internship program. The projects focus on Python concepts, problem-solving, and real-world applications.

## Task 1: Hangman Game

A text-based game where the user guesses a hidden word letter by letter.
Limited attempts (6 wrong guesses) Uses loops and conditional statements

## Task 2: Stock Portfolio Tracker

A program to calculate total investment value.
User inputs stock name and quantity Uses predefined stock prices Displays total value

## Task 3: Chatbot

A rule-based chatbot.
Responds to basic inputs like "hello", "how are you", "bye" Uses if-elif conditions

## Technologies Used

Python programming concepts (loops, conditions, functions)
